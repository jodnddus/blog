# 자료

## A Tour of Go

Go 언어 공식 홈페이지의 튜토리얼입니다.<br>
웹 브라우저에서 튜토리얼이 진행됩니다. 각 콘텐츠에서 다루는 내용들에 대해서 바로바로 코드를 작성해 볼 수 있다는 점이 큰 장점입니다. 또한 웹 브라우저에서 진행되기 때문에 개발 환경을 따로 세팅할 필요가 없습니다.
분량도 많지 않아 길게잡고 하루이틀이면 볼 수 있다고 생각합니다.

[A Tour of Go](https://go-tour-kr.appspot.com)

## 가장 빨리 만나는 Go 언어

Go 언어 책입니다. 전자책으로도 구매할 수 있습니다.<br>
이 책은 중간중간에 있는 "핵심정리" 부분을 제외하고 모든 부분을 무료로 볼 수 있는 부분 무료 책입니다.
Go 언어에 대한 이야기부터 기본 문법과 gorutine과 채널같은 고급문법 그리고 자잘한 프로젝트를 다루는 이 책은 보다 정돈된 느낌의 학습을 하고 싶은 사람들에게 추천하고 싶습니다.
가만히 책만 보고있는 것에 싫증을 느끼시거나, 실전 프로젝트에 빨리 써보고 싶은 마음을 가지고 계신 분들은 이 책이 'A Tour of Go' 보다 지겹게 느껴지실 수 있다고 생각합니다.

[가장 빨리 만나는 Go 언어](http://pyrasis.com/go.html)

## Go 언어 웹 프로그래밍 철저 입문

이것도 Go 언어 책이고 전자책으로도 구매할 수 있습니다.<br>
"가장 빨리 만나는 Go 언어" 와 문법에 대한 내용 구성은 비슷합니다. 하지만 병행처리나 에러처리, 패키지같은 고급 개념에 있어서 조금 더 깊이를 느낄 수 있다고 생각합니다.
또한 책 막바지에는 웹 프레임워크, 마이크로 프레임워크, 블로그 만들기 프로젝트를 통해서 배운내용을 써먹어 볼 수 있게 구성해 놓았습니다.

개인적으로 프로젝트를 만들어보는 부분이 정말 재미있었습니다.

[Go 언어 웹 프로그래밍 철저 입문](https://thebook.io/006806/)

# 프로젝트

다른 사람들이 운영하는 프로젝트의 코드들을 읽어보며 연구하는 것도 엄청난 도움이 되는 것 같습니다.

저는 현재 gorilla/mux라는 Go언어 라우팅 라이브러리 프로젝트의 코드를 하나하나 읽어보고 있습니다. 이해가 되는 코드들을 만나면 재미있고, 이해가 되지 않는 코드들을 보면 검색해서 알아보는 재미를 느끼고 있습니다.

## Go언어로 오픈소스 배송조회 서비스 만들기

Go언어로 웹 개발하는 과정, 테저트하는 과정을 볼 수 있을 뿐만 아니라 오픈소스 프로젝트를 운영하는 과정을 엿볼 수 있습니다.
제게는 많은 궁금증이 풀리는 꿀같은 블로그 포스팅이였습니다.

[Go언어로 오픈소스 배송조회 서비스 만들기](https://subicura.com/2016/06/13/start-go-shipment-tracking-opensource.html)

## Mux

Go언어로 웹 프로그래밍을 할 때 사용되는 라우팅 라이브러리입니다.

깃허브: [gorilla/mux](https://github.com/gorilla/mux)
gorilla 공식 사이트: [mux](https://www.gorillatoolkit.org/pkg/mux)

## 그 외

Go 언어로 개발할 때 사용되는 라이브러리를 살펴보면 좋은 오픈소스 프로젝트를 많이 발견할 수 있습니다.

[unrolled/render](https://github.com/unrolled/render)
[urfave/negroni](https://github.com/urfave/negroni)
